## 手搓编译

1. `cmake -B build -DCMAKE_PREFIX_PATH=D:/Qt6.8.3/6.8.3/mingw_64/ -DCMAKE_BUILD_TYPE=RelWithDebInfo -G "MinGW Makefiles"`
2. `cmake --build build`



### cmake gui编译

Qt6_DIR和QT_DIR都是一个；路径，直接在表格中填写即可

```
D:/Qt6.8.3/6.8.3/mingw_64/lib/cmake/Qt6
```
